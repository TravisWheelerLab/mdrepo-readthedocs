# MDRepo TOML/Metadata Specification

Use the [metadata creation](https://mdrepo.org/metadata) page to generate a valid TOML/metadata file, or craft one by hand using the following guidance.

## Required Fields

A minimal [TOML](https://toml.io/en/) file must contain the following required fields:

* `lead_contributor_orcid`: The [ORCID](https://orcid.org/) (in the format _xxxx-xxxx-xxxx-xxxx_) of the contributor responsible for uploading the simulation.
* `trajectory_file_name` [text]: Trajectory files contain atomic coordinates of sequential snapshots of the simulation system. Examples of extensions of trajectory files generated by different popular MD simulation packages are GROMACS (_.xtc_/_.trr_), AMBER (_.netcdf_/_.crd_), CHARMM/NAMD (_.dcd_)
* `structure_file_name` [text]: A structure file contains the atomic coordinates of one snapshot of the simulation system. The most popular format is a PDB file. Examples of extensions of structure files, other than PDB formated, generated by different popular MD simulation packages are GROMACS (_.gro_), CHARMM (_.cor_).
* `topology_file_name` [text]: The topology file contains the parameters for how the atoms in the molecule interact with each other through bonded and non-bonded interactions, as well as any added constraints or exclusions. Examples of extensions of topology files generated by different popular MD simulation packages are GROMACS (_.top_/_.tpr_), AMBER (_parm*.dat_), CHARMM/NAMD (_.psf_).
* `temperature_kelvin` [integer]: The temperature of the simulation in the range 275-700 Kelvin.
* `integration_timestep_fs` [integer]: The integration time step used for the simulation in the range 1-5 fs
* `short_description` [text]: A short (300 characters) description of the simulation
* `software_name` [text]: Software name, e.g. ACEMD, AMBER, CHARMM, GROMACS, NAMD, SPONGE
* `software_version` [text]: Software version

## Optional Fields

The following fields are optional but users are _strongly encouraged_ to provide them, if applicable:

* `pdb_id` [text]: An associated Protein DataBank (PDB) ID
* `uniprot_ids` [array of text]: Associated Uniprot IDs

The following fields are entirely optional:

* `description` [text]: A longer description of the simulation
* `run_commands` [text]: The commands used to produce the simulation
* `dois` [array of text]: A list of digital object identifiers (DOI) (in leiu of [[papers]])
* `forcefield` [text]: Type of force field used to produce the simulation, e.g., AMBER, Amber ff99SB-ILDN, CHARMM36
* `forcefield_comments` [text]: Comments relating to the force field used, e.g., ligand parameters
* `protonation_method` [text]: The method used for adding the protonation state, e.g., PropKa, AutoDockFR, AutoDockFR
* `alias` [text]: A unique, user-assigned alias/accession for the simulation

## Optional Tables

The following [TOML tables](https://toml.io/en/v1.1.0#table) are optional:

* `[water]` (may be omitted, but all fields required if present)
    * `model` [required, text]: Water model used for the simulation, e.g., TIP3P, TIP4P
    * `density_kg_m3` [required, floating-point number]: Water density in the simulation in the range 900-1100 kg/m3
* `[[additional_files]]` (can be repeated)
    * `file_name` [required, text]: File name
    * `file_type` [required, text]: Additional file type, e.g., Input, Structure, Trajectory, Topology, Periodic boundary condition, Restart, Logs, Checkpoint, Parameters, User defined file, Miscellaneous
    * `file_description` [optional, text]: Comments or description of the file
* `[[ligands]]` (can be repeated)
    * `name` [required, text]: Name of the ligand
    * `smiles` [required, text]: SMILES string for the ligand
* `[[solutes]]` (can be repeated)
    * `name` [required, text]: Solute name, e.g., Cl-, K+, Na+, Phosphoric acid, Urea,
    * `concentration_mol_liter` [required, floating-point number]: Solute concentration in the range 0-1 mol/liter
* `[[papers]]` (can be repeated, or use dois)
    * `title` [required, text]: Paper title
    * `authors` [required, text]: Paper authors
    * `journal` [required, text]: Paper journal
    * `year` [required, integer]: Paper published year
    * `volume` [required, integer]: Paper volume
    * `number` [optional, text]: Paper number
    * `pages` [optional, text]: Paper pages
    * `doi` [optional, text]: Paper DOI
* `[[external_links]]` (can be repeated)
    * `url` [required, text]: A URL
    * `label` [optional, text]: The label to display
* `[[contributors]]` (can be repeated)
    * `name` [required, text]: Name of the contributor
    * `orcid` [optional, text]: ORCID of the contributor
    * `email` [optional, text]: Email address of the contributor
    * `institution` [optional, text]: Institution of the contributor
